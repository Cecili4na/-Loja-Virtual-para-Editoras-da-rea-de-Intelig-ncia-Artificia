# COMPIA E-commerce

Plataforma de e-commerce da COMPIA Editora para venda de livros de Inteligência Artificial.

## Stack Tecnológica

- Next.js 14
- React 18
- TypeScript
- Deploy: Vercel

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## Deploy na Vercel

1. Faça push do código para o GitHub
2. Conecte o repositório na Vercel
3. Deploy automático!
